### About me

```yaml
name:       Egor Voronianskii
role:       Senior Java Developer / Tech Lead
domain:     Fintech · Banking systems
focus:      Microservices · Fault-tolerance · Event-driven architecture
languages:  Russian (native) · English (C1)
education:  B.Sc. Computer Science — Sunway University, Malaysia
```

I design and build high-load banking platforms in fintech — microservice architectures (~90 services scale),
fault-tolerant integrations, and event-driven workflows on Camunda + Kafka.
I lead engineering teams, run code reviews, and care a lot about API-first design,
observability, and code quality.

When I'm offline from Java, I write — on [Medium](https://vrnsky.medium.com),
[Substack](https://vrnsky.substack.com), and [Ghost](https://vrnsky.ghost.io) —
about the JVM, Spring internals, distributed systems, and the craft of software engineering.
